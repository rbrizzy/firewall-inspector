# Legacy / archived documents

These files describe an earlier, **abandoned** concept — *Packet Quest*, a network
**routing** adventure — that predates this project. They are **not** the design of
the game in this repository (**Firewall Inspector**, a firewall-verdict game).

They are kept only because their tech-stack and architecture choices (no build
step, single `index.html`, pure `engine/` + DOM/SVG `views/`, `localStorage`,
vendored THREE.js) were carried forward and reinterpreted for Firewall Inspector.

- [`PLAN.md`](./PLAN.md) — Packet Quest technical plan (archived)
- [`SPEC.md`](./SPEC.md) — Packet Quest specification (archived)
